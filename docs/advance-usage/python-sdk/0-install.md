# 安装

***如何安装***

## 下载

```
git clone https://github.com/primihub/primihub.git
cd primihub/python
```

## 编译

    见(代码编译)[https://docs.primihub.com/docs/developer-docs/build]

## 安装

1. 执行 `pip3 install -r requirements.txt` ，

2. 安装 primihub， 执行 `python setup.py install --user` ，
